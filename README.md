# Photo-storage
A project to work with data from the network, with the implementation of saving data to a database, search for photos on request with pagination

*[API](https://unsplash.com/documentation#search-photos)*
# Stack
+ Swift
+ UIKit
+ MVP
+ Realm
+ URLSession
